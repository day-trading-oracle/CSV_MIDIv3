MusicMidi - Text to MIDI Converter

A Python-based tool that converts text-based musical notation into MIDI files.

Quick Start:
1. Install dependencies: pip install -r requirements.txt
2. Run setup script: ./setup.ps1
3. Place your song files in the input/ directory
4. Run: python midi_generator.py input/your_song.txt

Features:
- Convert text notation to MIDI
- Support for single notes and chords
- Dynamic control
- Multiple duration types
- Precise note timing
- Automatic file versioning

For detailed documentation and examples, see README.md

Project Structure:
- midi_generator.py: Main conversion script
- input/: Place song files here
- output/: Generated MIDI files
- setup.ps1: Setup script
- requirements.txt: Python dependencies
