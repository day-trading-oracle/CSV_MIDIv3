# Variation Principles (Rooted in Music Theory & Drumming Practice)

# 1. Groove Pocket Adjustment
# - Concept: Shift the "pocket" (feel) of the groove by altering timing slightly, a technique used by drummers like Questlove for funk/soul.
# - Method: Push or pull hits by small increments (e.g., ±0.25 or ±0.125 beats) to create a laid-back or rushed feel.
# - Example: 0.0|q|4|kick 2.0|q|5|snare -> 0.25|q|4|kick 2.25|q|5|snare (laid-back).
# - Theory: Mimics swing or humanized timing; adjust within genre (e.g., subtle for Rock, pronounced for Funk).
# - Variations per Pattern: 2 (push, pull).
# - Total: 40 × 2 = 80 potential.

# 2. Syncopation Enhancement
# - Concept: Add or remove off-beat hits to increase rhythmic complexity, common in Jazz/Funk (e.g., Steve Gadd).
# - Method: Insert syncopated notes (e.g., kick or snare on 1.5, 2.5) or delete on-beat hits for space.
# - Example: 0.0|q|4|kick 2.0|q|5|snare -> 0.0|q|4|kick 1.5|e|4|snare 2.0|q|5|snare.
# - Theory: Syncopation drives groove tension/release; align with genre (e.g., sparse in Blues, dense in Funk).
# - Variations per Pattern: 2 (add, remove).
# - Total: 40 × 2 = 80 potential.

# 3. Ghost Note Layering
# - Concept: Add quiet snare or hihat "ghost notes" for texture, a staple in Hip-Hop/R&B (e.g., J Dilla).
# - Method: Insert low-dynamic (y=1 or 2) snare/hihat hits between main beats (e.g., 0.5, 1.5).
# - Example: 0.0|q|4|kick 2.0|q|5|snare -> 0.0|q|4|kick 0.5|e|2|snare 2.0|q|5|snare.
# - Theory: Enhances groove depth; common in swung or sixteenth-note patterns.
# - Variations per Pattern: 2 (snare ghosts, hihat ghosts).
# - Total: 40 × 2 = 80 potential.

# 4. Cymbal & Fill Accents
# - Concept: Use crash/ride or tom fills to punctuate or transition, as in Rock/Metal (e.g., Neil Peart).
# - Method: Add crash on downbeats (e.g., 0.0) or replace hihat with ride; append tom fills (e.g., 3.5-4.0).
# - Example: 0.0|q|4|kick 2.0|q|5|snare -> 0.0|q|4|kick 0.0|q|5|crash 2.0|q|5|snare 3.5|e|4|toms 3.75|e|4|toms.
# - Theory: Adds dynamic contrast; genre-specific (e.g., crash in Rock, ride in Jazz).
# - Variations per Pattern: 3 (crash, ride, tom fill).
# - Total: 40 × 3 = 120 potential.

# 5. Polyrhythmic Overlay
# - Concept: Introduce a secondary rhythm (e.g., 3-over-4) for complexity, used in Prog/Jazz (e.g., Danny Carey).
# - Method: Overlay triplet or dotted rhythms (e.g., kick every 1.33 beats) over the base 4/4.
# - Example: 0.0|q|4|kick 2.0|q|5|snare -> 0.0|q|4|kick 1.33|e|4|kick 2.0|q|5|snare 2.66|e|4|kick.
# - Theory: Polyrhythms create tension; best for advanced genres (Jazz, Metal).
# - Variations per Pattern: 1 (triplet overlay).
# - Total: 40 × 1 = 40 potential.

# Implementation Strategy
# - Start with 40 base patterns from Drum_Patterns.txt.
# - Apply principles to balance variety and genre fit:
#   1. Groove Pocket: 15 patterns × 2 = 30 (laid-back for Blues/Funk, rushed for Rock).
#   2. Syncopation: 10 patterns × 2 = 20 (add for Funk/Jazz, remove for Reggae/Pop).
#   3. Ghost Notes: 10 patterns × 2 = 20 (snare for Hip-Hop, hihat for R&B).
#   4. Cymbal/Fill: 10 patterns × 3 = 30 (crash for Rock, ride for Jazz, toms for Metal).
#   5. Polyrhythm: 5 patterns × 1 = 5 (Prog/Jazz focus).
# - Total: 30 + 20 + 20 + 30 + 5 = 105; trim to 100 by removing least distinct (e.g., minor pocket shifts).
# - Combine principles sparingly (e.g., Ghost + Syncopation) for extra flavor if needed.
